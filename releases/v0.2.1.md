# v0.2.1

Patch release for the `health` suite.

## What Changed

- added machine-checked reply validators for `health-review`, `doctor-brief`, `health-reminders`, and `health-storage-feishu`
- updated those flows so saved paths, due reasons, and bundle targets are explicit in generated output
- wired the new validators into suite acceptance and installed-workspace verification
- refreshed stable install docs from `v0.2.0` to `v0.2.1`
