# Roadmap

| Stage | Status | Goal | Exit Criteria | Unlocks |
| --- | --- | --- | --- | --- |
| [Stage 1](reference/openclaw-skills/development-plan.md#stage-1-workspace-and-local-first-baseline) | done | Establish the workspace and the local-first health baseline | Root docs, architecture, control surface, `health` landing docs, and the local data contract all exist | Health can be delivered as a first-class skill set |
| [Stage 2](reference/openclaw-skills/development-plan.md#stage-2-archive-and-family-doctor-baseline) | done | Deliver the first archive and doctor loop | `health-archive`, `private-doctor`, local workspace scripts, and GitHub release/install entry all work at baseline level | Trustworthy intake plus basic family-doctor dialogue |
| [Stage 3](reference/openclaw-skills/development-plan.md#stage-3-health-suite-install-and-doctor-core-completion) | active | Close the core capability gap: full-suite install, proactive onboarding, initial risk assessment, and post-image doctor dialogue | One command installs the full `health` suite; `private-doctor` completes onboarding, baseline assessment, and first-phase planning; CLI end-to-end tests prove image archive -> doctor dialogue | The old health agent can be reset and rebuilt from the skill suite |
| [Stage 4](reference/openclaw-skills/development-plan.md#stage-4-longitudinal-review-and-clinician-briefs) | planned | Deliver long-term review, summary, and clinician-facing outputs | `health-review` and `doctor-brief` generate daily/weekly/monthly reviews and clinician-readable briefs from the local workspace | The suite supports sustained follow-up and clinic prep |
| [Stage 5](reference/openclaw-skills/development-plan.md#stage-5-reminders-migration-and-health-v1-closeout) | planned | Deliver reminders, reset/migration readiness, and health V1 closeout | Reminder flow, backup/reset playbook, and CLI acceptance for install, archive, dialogue, review, and brief all exist; Feishu remains optional | The intended health agent V1 capability is fully covered by skills |
